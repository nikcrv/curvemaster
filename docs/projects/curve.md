# Curve Finance

Основной протокол экосистемы Curve Finance. Предоставляет эффективные обмены стейблкоинов и похожих активов с минимальным проскальзыванием.

## Основные функции

- Обмены стейблкоинов с низкими комиссиями
- Пулы ликвидности для различных активов
- Система голосования и управления через CRV токен
- Блокировка токенов в veCRV для участия в управлении

## Интеграции

- **Convex Finance**: Оптимизация доходности от Curve LP токенов
- **Frax Finance**: Партнерство по ликвидности стейблкоинов
- **crvUSD**: Нативный стейблкоин протокола
- **Yearn Finance**: Стратегии доходности на базе Curve
- **LlamaLend**: Кредитование под залог Curve LP токенов
